# Differential Forms

## Scope

Coordinate-free calculus based on alternating tensors and the exterior
derivative, bridging geometry, topology, and analysis.

## Core Topics

- Exterior algebra, wedge product, alternating multilinearity
- Differential forms on manifolds and pullback under smooth maps
- Exterior derivative and Cartan's magic formula framework
- Closed/exact forms and local-to-global structure
- Hodge star, codifferential, and Hodge Laplacian
- de Rham cohomology and topological invariants

## Canonical Results

- Poincaré Lemma
- de Rham Theorem (cohomology correspondence)
- Hodge decomposition (compact oriented Riemannian setting)

## Typical Applications

- Recasting vector identities in invariant form
- Computing topological obstructions via cohomology classes
- Variational PDEs with geometric operators
