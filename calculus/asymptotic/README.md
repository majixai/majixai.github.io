# Asymptotic Methods

## Scope

Approximation techniques for regimes with large/small parameters and multiscale
behavior.

## Core Topics

- Regular perturbation expansions
- Singular perturbations and boundary layers
- Matched asymptotic expansions
- WKB and eikonal transport hierarchy
- Laplace method and stationary phase
- Multiple scales and averaging

## Canonical Results

- Poincaré asymptotic expansions
- Boundary-layer matching principles
- Leading-order oscillatory integral asymptotics

## Typical Applications

- High-frequency wave propagation
- Reaction-diffusion and fluid boundary-layer analysis
- Semiclassical approximations in quantum models
