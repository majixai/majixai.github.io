# Exterior Calculus and Cartan Framework

## Scope

Higher-level differential calculus on manifolds using Lie derivatives,
connections, and moving frames.

## Core Topics

- Interior product $\iota_X$, Lie derivative $\mathcal{L}_X$, Cartan identity
- Principal bundles and connection 1-forms
- Curvature 2-forms and structure equations
- Covariant exterior derivative and Bianchi identities
- Differential invariants under symmetry groups

## Canonical Results

- Cartan's first and second structure equations
- Bianchi identities
- Frobenius integrability criterion

## Typical Applications

- Curvature computations in Riemannian/Lorentzian geometry
- Gauge-theoretic formulations of field equations
- Integrability and foliations in geometric mechanics
