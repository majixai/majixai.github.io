# Complex Analysis

## Scope

Holomorphic function theory from local analyticity to contour methods and
conformal geometry.

## Core Topics

- Cauchy-Riemann equations and holomorphicity criteria
- Cauchy integral theorem and integral formulas
- Laurent series and isolated singularities
- Residues and contour integration
- Meromorphic functions and argument principle
- Conformal maps and Riemann mapping theorem basics

## Canonical Results

- Morera and Liouville theorems
- Maximum modulus principle
- Open mapping theorem
- Residue theorem

## Typical Applications

- Evaluation of real integrals by contour deformation
- Asymptotics via steepest-descent contours
- 2D potential flow and conformal coordinate transforms
