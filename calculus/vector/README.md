# Vector Calculus

## Scope

Differential and integral calculus of vector fields in Euclidean spaces and
coordinate systems used in physics and engineering.

## Core Topics

- Differential operators: gradient, divergence, curl, Laplacian
- Identities: curl-grad zero, div-curl zero, vector Laplacian decompositions
- Line, surface, and volume integrals with orientation conventions
- Green, Stokes, and divergence theorems as one unifying framework
- Orthogonal curvilinear coordinates and scale factors
- Conservative fields and path-independence criteria

## Canonical Results

- Helmholtz decomposition (under suitable decay/boundary assumptions)
- Green's identities and weak formulations
- Potential theory basics and harmonic functions

## Typical Applications

- Electromagnetism and fluid mechanics identities
- Flux-conservation derivations
- Boundary value formulations for Poisson/Laplace problems
