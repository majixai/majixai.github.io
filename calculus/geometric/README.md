# Geometric Calculus

## Scope

Calculus on Riemannian manifolds with geometric operators, curvature, and
comparison principles.

## Core Topics

- Levi-Civita connection and covariant differentiation
- Geodesics, exponential map, injectivity radius concepts
- Hessian/Laplacian comparison heuristics
- Ricci/sectional curvature interactions with analysis
- Bochner technique and gradient estimates

## Canonical Results

- Hopf-Rinow theorem (metric completeness implications)
- Bochner-Weitzenböck identities
- Selected comparison theorems (Bishop-Gromov, Myers-style consequences)

## Typical Applications

- Heat kernel and eigenvalue estimate strategies
- Rigidity/splitting phenomena under curvature bounds
- Geometric PDE estimates on manifolds
