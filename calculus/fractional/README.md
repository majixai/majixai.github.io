# Fractional Calculus

## Scope

Non-integer differentiation/integration operators for memory-dependent and
nonlocal dynamical systems.

## Core Topics

- Riemann-Liouville and Caputo derivatives
- Grünwald-Letnikov discretization viewpoint
- Fractional Laplacian and nonlocal operators
- Mittag-Leffler functions and fractional ODE solutions
- Well-posedness principles for fractional evolution equations

## Canonical Results

- Semigroup-like properties and caveats for fractional operators
- Laplace-transform solution formulas for linear fractional systems
- Maximum principles in selected fractional PDE settings

## Typical Applications

- Anomalous diffusion and transport
- Viscoelastic constitutive modeling
- Long-memory stochastic and control models
