# Multivariable Calculus

## Scope

Advanced multivariable differential and integral calculus in finite dimensions,
with emphasis on structural theorems used in geometry, PDEs, and optimization.

## Core Topics

- Fréchet and Gâteaux differentiability in Banach-space notation
- Jacobian calculus, rank conditions, and constant rank theorem
- Implicit and inverse function theorems with quantitative estimates
- Higher-order derivatives, symmetric multilinear forms, and jets
- Multivariate Taylor expansions with integral and Peano remainders
- Change-of-variables, area and coarea formulas
- Sard's theorem and transversality heuristics

## Canonical Results

- Inverse Function Theorem
- Implicit Function Theorem
- Constant Rank Theorem
- Morse Lemma (local normal form near non-degenerate critical points)

## Typical Applications

- Local parameterizations of constraint manifolds
- Stability analysis of nonlinear systems
- Derivation of local normal forms for optimization landscapes
- Coordinate transforms in high-dimensional integration
