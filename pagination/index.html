1 · Table Pagination (DOM mode — Paginator.fromTable)

Existing <tbody> rows are hidden/shown without re-rendering.

#TickerSectorPrice

2 · List Pagination (DOM mode — Paginator.fromList)

Works with any <ul> or <ol>.

3 · Card Grid Pagination (DOM mode — Paginator.fromCards)

Paginates any container whose children are cards.

4 · Data-Array Mode (custom render callback)

Pass a data array and a render(item, container) function. The container is cleared and re-rendered on each page change.

5 · Dark Theme (.pgr-dark)

Add pgr-dark class to the .pgr-nav element or any ancestor.

6 · Integration Guide

Include in any page

<link rel="stylesheet" href="/pagination/pagination.css">
<script src="/pagination/pagination.js"></script>

Paginate a data array

const pager = new Paginator({
  data:         myArray,
  container:    '#my-list',
  paginationEl: '#my-pg',
  perPage:      10,
  render(item, container) {
    container.insertAdjacentHTML('beforeend', `<li>${item.name}</li>`);
  },
  onPageChange(page, items) {
    console.log('Page', page, 'showing', items.length, 'items');
  },
});

Paginate an existing <tbody>

// Rows already in DOM — no re-render needed
const pager = Paginator.fromTable('#my-tbody', {
  paginationEl: '#my-pg',
  perPage: 10,
});

Paginate a list

const pager = Paginator.fromList('#my-ul', {
  paginationEl: '#my-pg',
  perPage: 8,
});

Paginate a card grid

const pager = Paginator.fromCards('#my-cards', {
  paginationEl: '#my-pg',
  perPage: 12,
});

URL hash state persistence

const pager = new Paginator({
  data:         rows,
  paginationEl: '#pg',
  container:    '#list',
  perPage:      20,
  urlHash:      true,   // page survives browser reload / back button
  hashParam:    'page', // URL becomes #page=3
  render(item, el) { /* ... */ },
});

Dynamic updates

pager.setData(newArray);    // replace data, reset to page 1
pager.setPerPage(20);       // change items per page
pager.goTo(3);              // jump to page 3
pager.next();               // next page
pager.prev();               // previous page
pager.destroy();            // clean up event listeners
