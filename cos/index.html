Direction Cosines & Axis Angles

For an n-dimensional vector v, the direction cosines are the components of the unit vector v̂ = v/|v| — equal to cos(θᵢ) where θᵢ is the angle from v to the i-th coordinate axis.

Gram / Pairwise Cosine Matrix

Enter one vector per line (comma-separated numbers). Returns the m×m matrix of all pairwise cosine similarities.

Weighted Cosine Similarity

Rescale each dimension by a weight before computing cosine similarity. Useful when dimensions have different importance.

Soft Cosine Similarity

Uses a feature-similarity matrix S to account for correlations between dimensions. softCos(a, b, S) = ⟨a,b⟩S / √(⟨a,a⟩S·⟨b,b⟩S).
