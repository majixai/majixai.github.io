Nightly Market Forecast

This directory is the GitHub-hosted source of truth for the Google Apps Script nightly forecast workflow.

Runtime

Google Apps Script sends the nightly 10 PM ET forecast email after checking whether the next morning is a US equities trading session.

Prompt + Manifest

Both the project manifest and the Gemini prompt are versioned in this repo so GitHub stays authoritative.

Artifacts

Compressed .dat.gz artifacts live in gas_market_forecast/artifacts/ and are maintained with Bash helpers.
