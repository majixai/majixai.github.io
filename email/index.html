MajixEmail

Shared email library — usable from any MajixAI directory

Quick Start

<!-- 1. Optional: load EmailJS if you use that provider -->
<script src="https://cdn.jsdelivr.net/npm/@emailjs/browser@4/dist/email.min.js"></script>

<!-- 2. Load email-core.js -->
<script src="/email/email-core.js"></script>

<!-- 3. Configure and initialise -->
<script>
  window.EMAIL_CONFIG = {
    provider:  'mailto',      // 'mailto' | 'emailjs' | 'formsubmit'
    toEmail:   'hello@example.com',
    subject:   'Hello from MajixAI',
  };
  MajixEmail.init();
</script>

API Reference

MethodDescription
MajixEmail.init(config?) Initialise with the given config (or window.EMAIL_CONFIG). Returns MajixEmail for chaining.
MajixEmail.send(payload) Send an email. Returns a Promise. payload: { to, from, fromName, subject, body, replyTo }
MajixEmail.validate(payload) Validate a payload. Returns an array of error strings (empty = valid).
MajixEmail.render(name, vars) Render a named template with {{var}} substitution. Returns { subject, body }.
MajixEmail.bindForm(el, opts?) Attach to a <form> element and send on submit. Reads name attributes.
MajixEmail.buildMailtoLink(payload) Build a mailto: URL string without opening it.
MajixEmail.getHistory() Return the sent-email history array from localStorage.
MajixEmail.clearHistory() Erase the sent-email history from localStorage.

Providers

ProviderHow it worksRequirements
mailto Opens the visitor's native mail client via a mailto: link None — works on every static site
emailjs Uses the EmailJS SDK to send from the browser without a server Free EmailJS account + serviceId, templateId, publicKey
formsubmit POSTs form data to FormSubmit.co Free FormSubmit account + verified toEmail

Template Example

window.EMAIL_CONFIG = {
  provider: 'mailto',
  toEmail:  'support@example.com',
  templates: {
    welcome: {
      subject: 'Welcome, {{name}}!',
      body:    'Hi {{name}},\n\nThanks for joining MajixAI.\n\nBest,\nThe Team',
    },
    reset: {
      subject: 'Password reset for {{email}}',
      body:    'Click the link to reset your password:\n{{link}}',
    },
  },
};
MajixEmail.init();

// Render and send a template
const msg = MajixEmail.render('welcome', { name: 'Alice' });
MajixEmail.send({ ...msg, to: 'alice@example.com' });

Form Binding

<!-- HTML -->
<form id="contactForm">
  <input name="name"    placeholder="Your name">
  <input name="email"   placeholder="Your email">
  <input name="subject" placeholder="Subject">
  <textarea name="message"></textarea>
  <button type="submit">Send</button>
  <span data-email-status></span>
</form>

<script>
MajixEmail.init({ provider: 'formsubmit', toEmail: 'you@example.com' });
MajixEmail.bindForm(document.getElementById('contactForm'));
</script>
