    About

      MajixManage

      A shared README processing library for majixai.github.io. Parse, render, edit, and update marker-delimited sections in any Markdown document — all in the browser or from the command line.

      Marker convention

      <!-- START_SECTION_NAME -->
…content…
<!-- END_SECTION_NAME -->

      Installation

      <link  rel="stylesheet" href="/manage/manage.css">
<script src="/manage/manage-core.js"></script>

      🔍 Parse

      Extracts all marker sections from any Markdown string, preserving positions and raw content.

      ✏️ Edit

      Set, add, or remove any section by name. All mutations are reversible via undo/redo.

      🖥️ Render

      Converts Markdown to HTML with headings, tables, code blocks, lists, images, and links.

      📡 Load / Save

      Fetch a remote README via URL and trigger browser downloads of the edited document.

      ⏮️ History

      Ring-buffer undo/redo with configurable limit and full audit trail.

      💾 Persist

      Auto-saves the current document to localStorage across page refreshes.

      ⬇️ Export

      Export the document as Markdown, HTML, or a JSON snapshot of all sections.

      🖥️ CLI

      Node.js CLI (manage-cli.js) for batch README processing, scanning, and stats.
